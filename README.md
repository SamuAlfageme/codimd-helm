# CodiMD Helm chart

This repository contains helm chart for CodiMD.

## Requirements

- Kubernetes: >= 1.14
- Helm 2 / 3

## Install

```bash
helm repo add codimd https://helm.codimd.dev/
helm install my-release codimd/codimd
```
